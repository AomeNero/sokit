# Sokit

Sokit 是一款功能强大的 TCP & UDP 数据包发送/接收/转发工具，使用 C++ 和 Qt 框架开发，支持 Windows 和 Linux 平台。

## 功能特点

- **多协议支持**：同时支持 TCP 和 UDP 协议
- **三种工作模式**：服务器（Server）、转发（Transfer）、客户端（Client）
- **消息列表面板**：支持创建、编辑、管理消息列表，实现批量发送
- **消息搜索**：支持在消息列表中按内容/描述搜索，快捷键导航匹配项（Enter 下一个，Shift+Enter 上一个）
- **500ms 步进对齐**：延迟设置支持 500ms 步进对齐，便于精确控制发送时间
- **循环发送**：支持消息循环发送，并显示循环轮次数
- **数据持久化**：消息列表自动保存到本地文件，下次启动时自动加载
- **智能复制**：复制日志时自动去除时间戳，非打印字符转为 `[XX]` 格式
- **批量删除策略**：日志达到一定数量后自动批量删除，减少频繁操作
- **友好的用户界面**：直观的操作界面，支持中文显示

## 支持平台

- Windows
- Linux

## 编译环境

### 依赖项

- **Qt 框架**：5.0 及以上版本
- **C++ 编译器**：
  - Windows: Visual Studio 2010 及以上
  - Linux: GCC 4.8 及以上

### 编译步骤

#### Windows 平台

1. 安装 Qt 开发环境（推荐使用 Qt Creator）
2. 打开 `sokit.pro` 项目文件
3. 选择目标构建配置（Debug/Release）
4. 点击 "构建" 按钮编译项目
5. 编译完成后，可在构建目录找到可执行文件

#### Linux 平台

1. 安装 Qt 开发环境：
   ```bash
   sudo apt-get install qt5-default qttools5-dev-tools
   ```
2. 进入项目目录，执行：
   ```bash
   qmake
   make
   ```
3. 编译完成后，可在当前目录找到可执行文件

## 使用方法

### 客户端模式

1. 选择协议类型（TCP/UDP）
2. 输入目标 IP 地址和端口
3. 点击 "连接" 按钮建立连接
4. 在数据输入框中输入要发送的数据
5. 点击 "发送" 按钮发送数据
6. 接收区会显示接收到的数据

### 服务器模式

1. 选择协议类型（TCP/UDP）
2. 输入监听端口
3. 点击 "监听" 按钮开始监听
4. 当客户端连接后，可在连接列表中看到连接的客户端
5. 选择要发送的客户端，输入数据并点击 "发送" 按钮

### 转发模式

1. 选择协议类型（TCP/UDP）
2. 配置源端（S）和目标端（D）的 IP 地址和端口
3. 点击 "启动" 开始转发
4. 数据将在源端和目标端之间双向转发

### 消息列表面板

1. 点击右侧面板的 `>>` / `<<` 按钮展开/折叠消息列表面板
2. 点击 "+" 按钮添加新消息
3. 编辑消息内容、描述和延迟时间
4. 勾选要发送的消息
5. 点击 "Send" 按钮发送选中的消息
6. 点击 "Loop" 按钮开启循环发送模式

### 消息搜索

在消息列表面板顶部的搜索框中输入关键词：
- 自动搜索 **数据（Data）** 和 **描述（Desc）** 两列
- 匹配的行以蓝色高亮显示
- 按 **Enter** 跳转到下一个匹配项
- 按 **Shift+Enter** 跳转到上一个匹配项
- 右侧显示当前匹配位置（如 `1/5`）

## 界面介绍

### 主界面

- **顶部标签页**：Server、Transfer、Client、记事本四个标签
- **左侧区域**：连接控制、数据收发日志（树形显示 + 详情文本框）
- **右侧面板**：消息列表面板（可展开/折叠）

### 消息列表面板

- **工具栏**：折叠、添加、删除、上移、下移、搜索框
- **复选框**：选择要发送的消息
- **数据**：消息内容
- **描述**：消息描述
- **序号**：消息发送顺序
- **延迟**：发送延迟时间（毫秒）
- **操作按钮**：Send（发送选中）、Loop（循环发送）

## 功能说明

### 500ms 步进对齐

在设置消息延迟时，点击上下箭头会自动对齐到最近的 500ms 倍数：
- 增加：向上取整到最近的 500ms 倍数（例如：1234 → 1500）
- 减少：向下取整到最近的 500ms 倍数（例如：1234 → 1000）

### 循环发送

开启循环发送模式后，消息会不断重复发送，每次循环开始时会显示当前循环轮次数。

### 智能复制

在日志区选中条目后复制（Ctrl+C），会自动：
- 去除时间戳和标题前缀，只复制数据内容
- 将非打印字符转换为 `[XX]` 格式

### 批量删除策略

当日志条目达到一定数量时，会自动批量删除较早的日志条目，减少频繁操作带来的性能损耗。
